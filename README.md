# 🔍 FindMeAI – Lost Person Detection System

## 📌 Overview

**FindMeAI** is an AI-powered system designed to assist in locating missing persons using computer vision and machine learning techniques. The system analyzes images or video streams, detects human faces, and matches them against a predefined dataset to identify potential matches.

This project aims to support real-world use cases such as surveillance, law enforcement assistance, and social welfare initiatives.

---

## 🚀 Features

* 🧠 Face Detection using computer vision
* 🔎 Face Matching with stored dataset
* 📷 Works with images / video input
* ⚡ Fast and efficient processing
* 📊 Scalable for larger datasets (future scope)

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries & Tools:** OpenCV, NumPy, Pandas
* **Machine Learning / AI:** TensorFlow / Face Recognition Models
* **Development Environment:** VS Code

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Ipsita-Kundu/find_me_ai.git
cd find_me_ai
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Project

* Open the notebook:

```bash
jupyter notebook
```

* Run `findmeai.ipynb`

---

## 🧪 How It Works

1. Input image/video is provided
2. Face detection algorithm identifies faces
3. Extracted features are compared with stored dataset
4. Matching result is displayed
